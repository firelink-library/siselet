## Perguntas de validação

Se esforce para responder a essas perguntas com suas próprias palavras. Busque
por essas informações no material de referência ou de qualquer outra fonte.

1. Se você precisasse categorizar os tipos de memória de um computador em dois
   grandes grupos, como você faria essa divisão?

2. Por quê uma memória RAM não usa registradores com flipflops tipo d? Qual é o
   ganho de se usar transistores e capacitores? Qual a desvantagem?

3. Os bits dentro de um SSD são quantizados com mais do que dois níveis
   distintos (0 e 1). Quantos níveis se usa em dispositivos modernos? Por que
   fazer isso?

4. O que acontece em caso de flutuação de tensão no sistema de armazenamento de
   bits de uma memória RAM? Há mecanismos para garantir que essa sobrecarga não
   aconteça?

5. Por quê SSDs são menores e mais confiáveis que HDDs? Por quê ainda se usa
   HDDs para amazenar dados digitais?

6. Organize em ordem crescente de velocidade de leitura e escrita e escreva em
   uma única frase como funciona cada método de armazenamento:
    * HDD
    * Memória RAM
    * CD
    * Cache L2
    * Cache L1
    * SSD
    * BluRay
    * Fita magnética
    * Cartão perfurado
